# Java Spring Boot App Redis Demo

A simple repository to show how to connect to a Redis instance from a Spring Boot app using the Redis Client Jedis library.

## Setup 
You need the following setup on your machine to get started with this repo
1. [Install Redis]
2. Launch the terminal, command prompt or Git bash
3. Start redis server
4. clone this repository
 ```
 git clone git@github.com:cptdanko/java-redis-jedis-demo.git
 ```
5. Navigate to the directory and run the app
 ```
 mvn spring-boot:run
 ```
7. Once you see the message
```
: Started RedisApplication in ....
```
9. Open postman
10. Import the .json file from the folder PostmanCollection from the directory of this repo and have fun :)
